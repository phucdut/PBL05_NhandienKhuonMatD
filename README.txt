+Để sử dụng phần mềm bạn cần:                 						    +
+-------------------------------------------------------------------------------------------+
+1. Máy có cài đặt sẵn python 3								    +
+	Nếu bạn chưa biết cách cài python 3 hãy tham khảo:				    +
+	https://codecute.com/python/huong-dan-cai-dat-lap-trinh-python-tren-windows-10.html +
+-------------------------------------------------------------------------------------------+
+2. Chạy file Library( Bạn hãy đổi đuôi .txt sang .bat)				    	    +
+	Nếu file Library.bat bị lỗi thì hãy cài đặt thư viện thủ công:			    +
+	Bật cmd và thực hiện các câu lệnh:	    					    +
+	pip install cmake								    +
+	pip install dlib								    +
+	pip install pandas								    +
+	pip install numpy								    +
+	pip install Pillow								    +				    
+	pip install opencv-python							    +
+	pip install opencv-contrib-python						    +
+	pip install face-recognition							    +
+	pip install db-sqlite3								    +
+	pip install PyQt5
	pip install win32gui
	pip install pywin32								    +
+-------------------------------------------------------------------------------------------+
+3. Chạy file setup.exe ra màn hình Desktop						    +
+	Sau khi chạy file bạn hãy tìm đến file TheBoss.exe để khởi động phần mềm            +
+	và sử dụng!									    +
